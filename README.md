# 쿠팡 카페 자동 포스팅 시스템 배포 가이드

본 프로그램은 쿠팡 상품 정보를 수집하여 AI로 홍보 문구를 생성한 뒤, 네이버 카페에 자동으로 포스팅하는 도구입니다.

## 1. 필수 환경 (Prerequisites)
*   **Python 3.10 이상**: [python.org](https://www.python.org/)에서 설치 (설치 시 'Add Python to PATH' 체크 필수)
*   **Google Chrome 브라우저**: 최신 버전 설치 권장
*   **Gemini API Key**: [Google AI Studio](https://aistudio.google.com/)에서 발급

## 2. 설치 및 실행 방법
1.  본 폴더의 내용을 적당한 위치에 압축 해제합니다.
2.  `실행하기.bat` 파일을 더블 클릭합니다.
    *   첫 실행 시 필요한 라이브러리를 자동으로 설치합니다.
    *   "Uvicorn running on http://0.0.0.0:8000" 메시지가 뜨면 성공입니다.
3.  웹 브라우저를 열고 `http://localhost:8000` 주소로 접속합니다.

## 3. 사용 가이드
1.  **계정 설정**: 처음 사용 시 네이버 ID/PW, Gemini API Key 등을 입력하고 '저장' 버튼을 누릅니다.
2.  **카페 프리셋**: 자주 사용하는 카페 ID와 메뉴 ID를 저장해두면 편리하게 불러올 수 있습니다.
3.  **포스팅**: 쿠팡 URL들을 입력하고 간격을 설정한 뒤 '자동 포스팅 시작'을 누릅니다.

## 4. 주의 사항
*   **보안**: 네이버 계정 정보는 사용자의 브라우저(LocalStorage)에만 저장되며 외부로 전송되지 않습니다.
*   **업데이트**: 크롬 브라우저 버전이 크게 업데이트되어 작동하지 않을 경우, `config.py`의 `CHROME_VERSION_MAIN` 값을 수정해야 할 수 있습니다.
*   **정책 준수**: 과도한 포스팅은 계정 제한의 원인이 될 수 있으므로 적절한 시간 간격(최소 5분 권장)을 유지하세요.
